# 会话运行文档导航

## 作用

这页是 Obsidian 导航入口，只负责把当前正式会话规范串起来。

## 先读顺序

1. [[a1-a9-session-brief]]
2. [[a1-a9-session-protocol]]
3. [[lookme-anchored-task-flow]]
4. [[lookme-runtime-runbook]]
5. [[lookme-runtime-cheatsheet]]
6. [[workspace/projects/AEdu/task-dispatch-playbook|AEdu Task Dispatch Playbook]]

## 使用边界

- `[[a1-a9-session-brief]]`：每次正式会话开始前必读
- `[[a1-a9-session-protocol]]`：会话级正式协议真源
- `[[lookme-anchored-task-flow]]`：按 A0-A9 执行的锚点流
- `[[lookme-runtime-runbook]]`：运行与验活操作手册
- `[[lookme-runtime-cheatsheet]]`：现场速查
- `[[workspace/projects/AEdu/task-dispatch-playbook|AEdu Task Dispatch Playbook]]`：把用户任务翻成派单

## 当前统一口径

- `A1-A9` 是会话级生命周期
- `A6-A9` 是当前会话内的收口与分支判断
- 每次会话收口都必须先做本地回写，再决定是否进入后续会话
- 正式 CE 评论由 commander 写，`lookme` 默认只做本地回写
