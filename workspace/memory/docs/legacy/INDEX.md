# Legacy Source Materials

## Purpose
- `legacy/` 只保留旧工作区、旧系统、旧项目迁移过来的源材料
- 这些文档保留证据价值，但不具备当前 `workbot` 总项目的默认解释权

## Rules
- 不把 `legacy/` 中的文档直接当作现行规则
- 若旧文档中有稳定事实、经验或结构值得保留，应提炼到 `memory/kb/**`
- 若只需要追溯来源、核对历史、审计迁移过程，再读取 `legacy/`
