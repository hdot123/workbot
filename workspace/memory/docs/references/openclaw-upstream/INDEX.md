# OpenClaw Upstream Source Materials

## Purpose
- 本目录收纳上游 OpenClaw 官方资料与产品说明文档
- 这些文档属于 source material，不属于当前 `workbot` 总项目的现行规则

## Rules
- 需要了解上游能力、协议、接口或约束时再读取
- 不直接把本目录内容当作当前项目的启动链或人格定义
- 若上游资料中有需要固化的稳定规则，应提炼写入 `memory/kb/**`
