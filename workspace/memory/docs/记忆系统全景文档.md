# 记忆系统全景文档

Status: source-material, absorbed-by-policy

## Overview
- `workbot` memory runs with a split between legal canonicals, project canonicals, and source-material docs.
- `project-map/INDEX.md` is the legality entry for deciding whether material can become formal truth.

## Layers
1. `project-map/` decides legality and registry state.
2. `memory/kb/global/` holds global canonical rules.
3. `memory/kb/projects/` holds project canonicals.
4. `memory/docs/` holds incoming raw or supporting source material.

## Current Gate
- `memory_hook_gateway.py` validates required canonical paths, project-map rules, and truth-basis integrity before treating memory context as ready.
- `validate_memory_system.py` is the lightweight local validator for this memory contract.
