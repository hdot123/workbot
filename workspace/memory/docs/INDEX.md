# Docs Index

## Purpose
- `memory/docs/` 只承载研究资料、外部资料、收件箱材料、legacy 参考和引用材料
- 不承载项目真相；项目真相统一写入 `memory/kb/`
- `memory/docs/` 默认是资料区，不是现行规则区

## Sections
- `research/`：项目调研、论文阅读、对标分析
  - 入口：`research/INDEX.md`
- `external/`：外部新闻、外部资料汇编
  - 入口：`external/INDEX.md`
- `inbox/`：待分类的临时文档入口
  - 入口：`inbox/INDEX.md`
- `legacy/`：旧工作区或旧系统吸收过来的历史源材料
- `references/`：长期参考但不作为 canonical 的资料
  - `references/openclaw-upstream/`：上游 OpenClaw 官方资料，统一作为 source material 保留
  - 入口：`references/INDEX.md`

## Reading Rule
- 优先阅读 `research/`、`references/` 和已明确指定的资料目录
- `legacy/` 和 `memory/docs/` 根部的大量单文件文档默认视为来源材料，不默认参与当前规则解释
- 若旧资料中存在仍然有效的事实，应提炼进 `memory/kb/**`
