# Memory Index

## Purpose
- `memory/` 是总项目的记忆与资料总入口
- 这里同时承载当前真相层、研究资料层、历史源材料层、日志层和长期记忆层

## Main Areas
- `short-index.md`：最短启动索引
- `kb/`：canonical 真相层
- `docs/`：研究、参考、历史源材料和收件箱
- `log/`：按日追加的事实流水
- `reflections/`：复盘与反思材料
- `actions/`：任务收件箱
- `system/`：系统运行相关材料

1. `../MEMORY.md`

4. `kb/INDEX.md`

## Rule
- 真相写 `kb/`
- 资料写 `docs/`
- 历史记录写 `log/`
- 长期稳定记忆从 `kb/longterm/` 进入
