# Projects Index

## Purpose
- `projects/` 只存放项目交付材料、报告、实施文档和产物
- `projects/` 不是真相层；项目真相统一写入 `memory/kb/projects/**`

## Current Areas
- `AEdu/`：业务项目相关产物区
  - 入口：`AEdu/INDEX.md`
- `opencli/`：平台项目相关产物区
- `app/`：轻量平台目录的产物区
- `agents/`：代理与角色体系相关产物区
- `skills/`：技能/能力包相关产物区
- `openclaw-molt/`：历史来源项目材料区
  - 入口：`openclaw-molt/INDEX.md`

## Rule
- 长文档、交付稿、实施报告可以放在这里
- 若存在对应 canonical，应在文件顶部标注 canonical 指针
- 需要沉淀为稳定规则或事实时，提炼回 `memory/kb/**`

## Notes
- 当前 `AEdu/` 与 `openclaw-molt/` 已有明确入口
- 其余目录当前仅作为结构保留，待未来真实交付材料进入
