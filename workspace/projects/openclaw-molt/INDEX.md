# OpenClaw Molt Project Artifacts Index

## Purpose
- 本目录保存从 `OpenClaw Molt` 工作区吸收来的历史项目产物、报告、脚本和交付材料
- 这些内容属于历史项目产物区，不是当前总项目 canonical

## Current Materials
- `clawrouter-*`：ClawRouter 相关报告、清单、迁移材料
- `multi-brand-*`：多品牌协议相关材料
- `node-11_*` / `node-22-*`：节点部署与测试报告
- `mirofish-*`：MiroFish 相关分析与部署材料
- `*.sh` / `*.yaml` / `*.html`：历史执行脚本与辅助文件

## Rule
- 需要追溯旧实施材料、历史交付稿或脚本时再读
- 若其中有需要长期保留的规则或事实，应提炼回 `memory/kb/**`
