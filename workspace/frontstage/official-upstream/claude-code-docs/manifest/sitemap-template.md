## 使用规则

- 这里记录“站点结构”，不是页面正文摘要
- 同一页面若出现在多个导航入口，只保留一条主记录，并在 `remarks` 标注别名入口
- 若页面存在版本差异，在 `remarks` 标记版本范围或时间点
