- 本文件放的是“基于既有知识整理的官方候选链接”，不是联网核验后的最终结果。

- 一旦某链接已确认跳转、失效、迁移或归档，必须更新状态并补充备注。

## 状态建议

- `unverified-live`：高概率官方入口，但未实时核验
- `verified`：已确认可达且为官方来源
- `redirected`：原链接跳转到其他官方页面
- `archived`：历史官方页面或旧路径
- `blocked`：访问受限，暂不能确认
