# Claude Code refs 索引

## 目录职责

本目录只用于存放 Claude Code 的官方来源证明。

允许内容：

- 官方入口页
- 官方页面 URL 清单
- 页面到来源的映射
- 访问前提说明（如是否需要登录）

禁止内容：

- 分析结论
- 二次提炼后的知识结论
- 最终整理稿
- promotion 后的下游判断

## 建议文件

- `official-links.md`：官方链接清单
- `page-source-map.md`：页面到来源映射

## 使用说明

- 这里是证据层，不是结论层
- 一条链接可对应多个 raw 文件，但必须能追溯
- 如页面发生跳转或迁移，要保留旧链接记录并标注状态
