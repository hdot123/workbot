# Claude Code 当前待核点

## 与来源相关

- `docs.anthropic.com/en/docs/claude-code` 是否仍为当前现行 Claude Code 栏目入口
- 是否存在独立的 Claude Code 产品页，而不只是文档栏目页
- 发行说明/变更日志是否有独立栏目，还是分散在其他官方页面中

## 与页面结构相关

- 安装、配置、权限、工作流、IDE 集成是否为独立页面，还是同页分节
- 是否存在登录后才可见的 Claude Code 页面
- 是否存在历史迁移页面需要保留旧 URL 作为旁证

## 与后续 promotion 相关

- 哪些页面能作为“根事实页面”优先进入 promotion
- 哪些内容只是示例或操作提示，不应提升为稳定知识
- 哪些页面变化频率高，需要单独标记 freshness

## 当前结论边界

在未联网核验前：

- 可以用模型既有知识提供高概率官方入口种子
- 不能把这些 URL 当作最终已验证事实
- 不能把候选路径直接当作最终页面地图
