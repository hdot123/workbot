# Round 02 Project Compatibility Map

## Round Goal
- 吞噬 `workspace/memory/kb/global/projects/**` 的兼容残留。
- 让旧项目 canonical 路径只保留兼容跳转与退役痕迹，不再承载现行项目事实。
- 继续收紧“唯一合法真相只在 `workspace/memory/kb/projects/**`”这一层级。

## Scope
- `workspace/memory/kb/global/projects/INDEX.md`
- `workspace/memory/kb/global/projects/workbot.md`
- `workspace/memory/kb/global/projects/AEdu.md`
- `workspace/memory/kb/global/projects/platform-capabilities.md`
- `workspace/memory/kb/global/projects/opencli.md`

## What This Round Does
- 把 `global/projects/` 明确为 compatibility-only 目录。
- 清除兼容 stub 中仍残留的现行事实、悬空入口和误导性正文。
- 只保留 active canonical 跳转、状态说明和历史最小痕迹。

## Exit Condition
- `global/projects/` 不再承载 active project canonical。
- `workbot.md`、`AEdu.md`、`platform-capabilities.md` 只保留兼容跳转说明。
- `opencli.md` 只保留退役历史痕迹。
- 不再存在把旧兼容层误读为现行项目真相的正文残留。

## Result
- `workspace/memory/kb/projects/**` 成为唯一合法项目 canonical。
- `workspace/memory/kb/global/projects/**` 继续存在，但仅作为非合法兼容层。
