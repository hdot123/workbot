# AEdu

## 定位
- `AEdu` 是 `workbot` 总记忆系统中的业务项目域。
- 它不是并列的第二套总记忆系统。
- `AEdu` 的流程规范、任务派发、阶段执行和交接规则，都是总记忆系统在该项目上的派生落地。

## 归属规则
- 项目稳定事实、边界和派生规则写入本文件或同层 project canonical。
- 项目执行面材料进入 `../../projects/AEdu/`。
- 项目外部研究、对比和来源材料进入 `../../docs/research/projects/AEdu/`。
- 如果 `AEdu` 的某条规则已经提升为跨项目通用规则，应再提炼到 `../global/`，而不是继续留在项目域。
