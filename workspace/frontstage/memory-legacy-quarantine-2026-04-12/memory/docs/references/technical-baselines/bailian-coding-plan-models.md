## ⚠️ 关键区别

### bailian/glm vs zai/glm

| 对比项 | bailian/glm | zai/glm | 差异 |
|--------|-------------|---------|------|
| 最大输出 | 16,384 | **128,000** | **8 倍** ✅ |
| 上下文 | 202K | 204K | 相近 |
| 视觉实现 | Skill | MCP | 不同机制 |

### 最大输出对比

| 模型 | 最大输出 | 适用场景 |
|------|---------|---------|
| zai/glm-5 / zai/glm-4.7 | 128,000 | 超长文档、完整项目 ✅ |
| bailian/qwen3.5-plus | 65,536 | 通用任务、中等文档 ✅ |
| bailian/kimi-k2.5 | 32,768 | 中等任务 |
| bailian/glm-5 / bailian/glm-4.7 | 16,384 | 短任务、对话 ⚠️ |

---

## 📝 最大输出 Tokens 影响

| 影响项 | 说明 |
|--------|------|
| **单次回答长度** | 输出达到上限后强制停止 |
| **长任务处理** | 大文件/长文档可能需要分段 |
| **交互效率** | 低输出限制需要多次"继续"交互 |
| **代码完整性** | 大文件可能无法一次性生成完整 |
