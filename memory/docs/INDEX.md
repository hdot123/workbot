## 用途

- `docs/` 存放研究材料、参考资料与过渡性修订内容。
- `docs/` 不作为 canonical；稳定结论应回写到 `memory/kb/`。
- 除根索引与经地图明确吸收的文件外，其余 `docs/**` 默认按 `incoming-raw` 处理。

## 主要区域

- `research/`：当前分析、调研与专题研究入口，入口为 `research/INDEX.md`。
- `references/`：长期参考资料与上游材料，入口为 `references/INDEX.md`。
- `corrections/`：勘误与临时修订，不应长期替代 canonical。

## 合法性规则

- `docs/**` 是否可被正式引用，以 `project-map/INDEX.md` 和 `project-map/ingestion-registry-map.md` 的裁决为准。
- 未被地图明确吸收的 `docs/**` 内容，不得作为合法真相。
- 已被裁决为 `retired` 的文件只保留历史证据价值。
