## 使用边界

- `[[a1-a9-session-brief]]`：每次正式会话开始前必读
- `[[a1-a9-session-protocol]]`：会话级正式协议真源
- `[[cmux-runtime-handbook]]`：当前 `cmux 5+1` 执行与验活手册
- `[[cmux-subagent-runtime-chain]]`：当前 runtime canonical truth
- `[[projects/AEdu/task-dispatch-playbook|AEdu Task Dispatch Playbook]]`：把用户任务翻成派单
- `lookme` 系列文档：历史参考，不作为当前正式运行入口

## 当前统一口径

- `A1-A9` 是会话级生命周期
- `A6-A9` 是当前会话内的收口与分支判断
- 每次会话收口都必须先做本地回写，再决定是否进入后续会话
- 正式 CE 评论由 commander 写，自动收尾默认只做本地回写
