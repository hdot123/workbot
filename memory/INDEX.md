# Memory KB Root Index

## Purpose
- `kb/` 存放长期真相、稳定规范、正式决策与可复用经验
- 这里的内容优先于 `docs/`，但仍应按层级区分 canonical、decision 与 lesson

## Canonical
- `project-map/`：合法入口登记、原料区状态与吞噬波次
- `global/`：跨项目长期规范、总记忆系统合同与共享边界
  - 入口：`global/INDEX.md`
  - 版本化规范：`global/versions/INDEX.md`
- `projects/`：项目 canonical 与项目域派生规则
  - 入口：`projects/INDEX.md`

## Decisions
- `decisions/`：正式决策与裁决记录
  - 入口：`decisions/INDEX.md`

## Lessons
- `lessons/`：经验教训、SOP 与可复用运行结论
  - 入口：`lessons/INDEX.md`

## Reading Order
- 先读 `../../project-map/INDEX.md`
- 先读 `global/`
- 再读 `projects/`
- 需要裁决时再读 `decisions/`
- 需要方法与踩坑结论时再读 `lessons/`

## Compatibility
- `global/projects/` 自 2026-04-11 起只保留为兼容跳转层，不再作为 active project canonical。
